Para utilizar o programa va, atraves do terminal, ate o diretorio com os arquivos
de codigo fonte Catalogo.h Catalogo.cpp, main.cpp e os arquivos catalogo.csv e
makefile. Em seguida digite no terminal "make" para gerar o executavel do programa.
Entao, digite "./catalogo catalogo.csv" para executar e selecione entre as opcoes
oferecidas pelo menu do programa. Ademais, pode-se utilizar o comando "make clean"
no terminal para deletar o executavel.

O arquivo .csv eh atualizado ao fim da execucao do programa de acordo com as acoes
realizadas pelo usuario, assim os dados dos filmes persistem no catalogo para a
proxima execucao do programa.
